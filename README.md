# spinny-wheel

## Description

This is a simple wheel spinning application that allows users to spin a wheel based on a list of strings and select one randomly. The application is built using vanilla JavaScript, HTML, and CSS.

## Running the Application

To run the application, simply open a http server in the root directory of the project. For example, you can use the following command to run a simple http server:

```
npx http-server -a localhost .
```
